### ◈ Student Registration System — Full-Stack

<div align="center">

[![Student Reg Repo](https://github-readme-stats.vercel.app/api/pin/?username=yogeshkamisetty&repo=student-registration&theme=tokyonight&hide_border=true&title_color=818CF8&icon_color=38BDF8&bg_color=0d1117&text_color=94a3b8)](https://github.com/yogeshkamisetty/student-registration)

</div>

> Complete **3-tier web application** — HTML frontend → Node.js/Express backend → MySQL database — with a live Admin Dashboard and full REST API.

```
 ┌─────────────────── Student Registration System ──────────────────────────┐
 │                                                                          │
 │  TIER 1 · PRESENTATION                                                   │
 │  ┌──────────────────────┐    ┌─────────────────────────────────────────┐ │
 │  │    index.html        │    │           admin.html                    │ │
 │  │    Student Panel     │    │           Admin Dashboard               │ │
 │  │  · Registration form │    │  · Live student table  (auto-refresh)   │ │
 │  │  · Client validation │    │  · Search by name / email / roll no.    │ │
 │  │  · fetch() to API    │    │  · Delete records  ·  Export CSV        │ │
 │  └──────────┬───────────┘    └──────────────────────┬──────────────────┘ │
 │             │ HTTP Request                           │ HTTP Request       │
 │             └──────────────────┬─────────────────────┘                   │
 │                                ▼                                         │
 │  TIER 2 · APPLICATION                                                    │
 │  ┌──────────────────────────────────────────────────────────────────────┐│
 │  │                  server.js  (Node.js + Express.js)                  ││
 │  │                                                                      ││
 │  │   POST   /register       →  Validate & persist student record        ││
 │  │   GET    /users          →  Return all students as JSON              ││
 │  │   DELETE /delete/:id     →  Remove record by primary key             ││
 │  └────────────────────────────────────────┬─────────────────────────────┘│
 │                                           │ mysql2 driver                │
 │  TIER 3 · DATA                            ▼                              │
 │  ┌──────────────────────────────────────────────────────────────────────┐│
 │  │  MySQL  ·  studentdb                                                ││
 │  │  students ( id · roll · firstname · lastname · father_name · dob    ││
 │  │             mobile · email · gender · department · course            ││
 │  │             city · address )                                         ││
 │  └──────────────────────────────────────────────────────────────────────┘│
 └──────────────────────────────────────────────────────────────────────────┘

  Stack  :  HTML 68%  ·  JavaScript 19%  ·  CSS 14%
  Backend:  Node.js + Express.js  ·  mysql2 driver
  DB     :  MySQL  ·  studentdb  ·  CRUD operations
```

**REST API Reference**

| Method | Endpoint | Request Body | Response |
|--------|----------|-------------|----------|
| `POST` | `/register` | `{ roll, firstname, email, dept, ... }` | `201 Created` |
| `GET` | `/users` | — | `[{ student objects }]` |
| `DELETE` | `/delete/:id` | — | `200 OK` |
